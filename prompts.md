# Prompts

## 1. Create prompts.md

Create a prompts.md in root to save all prompts in

## 2. Create CLAUDE.md with project conventions

Create a CLAUDE.md with following:
- Stack: HTML + Tailwind (CDN), vanilla JS (fetch only), Supabase backend/db, no npm/node
- Conventions: no comments, no unsolicited refactoring, async/await always
- Tailwind: utility classes only, grouped by layout > spacing > typography > colors > effects > states
- Mobile-first layout, reusable components
- Save every prompt to prompts.md
- Update CLAUDE.md when necessary
